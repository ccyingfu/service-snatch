# Icons

This directory should contain the following PNG files:

- icon-16.png (16x16 pixels)
- icon-48.png (48x48 pixels)
- icon-128.png (128x128 pixels)

You can create these icons using any image editor or online tool.
For now, Chrome will use a default icon.

## Quick Solution
Visit: https://www.favicon-generator.org/ or use any icon editor to create PNG icons.
